# AI-Coding-Museum

> 微型内容社区平台 - 面向 AI Coding 垂直领域的轻量级 UGC 社区

## 项目状态

- **版本**：v0.1.0
- **状态**：✅ 已完成
- **更新时间**：2026-03-01

## 部署地址

| 环境 | 地址 |
|------|------|
| 生产环境 | https://ai-coding-museum-production.up.railway.app |

## 功能概览

| 模块 | 功能 |
|------|------|
| 用户认证 | 注册、登录、会话管理（BetterAuth） |
| 内容发布 | 发帖、时间流浏览、支持图片/视频 |
| 互动功能 | 帖子点赞、评论（5层嵌套）、评论点赞 |
| 用户关注 | 关注/取关、关注列表、粉丝列表、关注时间流 |
| 对象存储 | R2 Presigned URL 前端直传 |

## 技术栈

| 组件 | 技术 |
|------|------|
| 框架 | NestJS |
| 数据库 | Neon (PostgreSQL) + Drizzle ORM |
| 认证 | BetterAuth |
| 存储 | Cloudflare R2 |
| 缓存 | Upstash Redis（预留） |

## 快速开始

```bash
# 安装依赖
npm install

# 配置环境变量
cp .env.example .env
# 编辑 .env 填入真实值

# 数据库迁移
npx drizzle-kit migrate

# 启动服务
npm run start
```
